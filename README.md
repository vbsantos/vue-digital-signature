# Desafio AVMB - Fullstack Dev

## Descrição

Este é um SPA que inclui autenticação de usuário, gerenciamento de repositórios e manipulação de envelopes digitais com documentos e signatários.

## Tecnologias

- Vite
- Vue.js
- Vue Router
- Pinia
- Bootstrap

## Como usar

```bash
docker compose up -d
```
